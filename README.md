以喜欢小祥为契机而诞生的mod。

目前进度：

    丰川祥子- 机制：压力/回忆 
    压力-可作为额外资源使用；也可给予敌人用以增伤。
    回忆-初始0费，打出次数越多费用越高。
    ·缺少专属事件；
    ·缺少一部分金卡；
    ·缺少专属遗物；
    ·专属药水待填充；
    ·可能要重做回忆和遗忘机制；

    千早爱音- 机制：音符/和弦
    音符-打出攻击/技能/能力/其他牌时，会产生对应音符。
    和弦-特定序列的音符会组成和弦，演奏和弦会产生各种效果。
    （实则狩猎笛）
